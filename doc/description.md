SlavicsGearIndex

# Description
*Maximum 4000 Characters*
Display of the currently engaged sprocket only on the rear derailleur. The derailleur index is taken universally from AntPlus (Shimano Di2, SRAM AXS,...).
The application is independent of the number of sprockets. The display of the number of teeth on the sprocket is optional in the application settings.
When changing gears to another sprocket, the color is gray. The outer sprockets are displayed in dark red.

Localization EN, DE, CZ, PL, SK, IT, FR, HU.

If there is a wrong translation, please forgive me and contact me with the correct expression (in English or Czech).

Tested on SRAM Transmission GX derailleur.

The application does not support front derailleur!

Display
* Top right - current number of teeth on the cassette - optional.
* Bottom right - shifting errors (max. 10sec per error) in the order Shift failures / Invalid inboard shifts / Invalid outboard shifts - Optional
* Bottom left - system battery status

Thanks Petr

Report bugs, problems or requests:

Display value
*    -- - No connect AntPlus.ShiftingStatus
*    xx - No exists AntPlus.ShiftingStatus
